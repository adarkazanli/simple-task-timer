This tool allows you to create a list of tasks with time goals in hours, and keep track of the amount of time that you spend in each task.
It has several options available for customisation.

Please share any suggestions you have, as I'm always looking to improve the app. Thanks!

Current Features:
- Any number of tasks
- Tasks with goals
- Tasks with with indefinite goals (no goal)
- Task descriptions
- Start, pause, reset, delete, edit tasks
- Progress bars
- Edit task names, time spent, and goal-times
- Desktop notifications
- Sound notifications (default and custom)
- Pie charts
- CSV Exporting
- Task syncing (work in progress)
